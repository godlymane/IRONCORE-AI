Quick Start
Three Steps to Your First Call

Get up and running in under two minutes. No SDK required — it's just REST.

1

Get Your API Key

Download IronCore AI from the Play Store. Open your Profile, tap Connected Agents, and generate your API key. Keys start with ic_.

2

Make Your First Call

Fetch your profile with a single curl command:

bash 
curl -H "Authorization: Bearer ic_YOUR_KEY" \
  https://us-central1-ironcore-f68c2.cloudfunctions.net/apiGetProfile
3

Parse the Response

You'll get back your full fitness profile as JSON:

json 
{
  "username": "trex_",
  "level": 12,
  "xp": 5840,
  "league": "Gold",
  "currentForge": 14,
  "wins": 23,
  "losses": 8,
  "eloRating": 1456
}

Security
Authentication

All requests require a Bearer token. Keys are scoped to the permissions the user explicitly approved.

Bearer Token Required

All requests must include a valid API key in the Authorization header. Keys are generated per-user in the IronCore app.

Scoped Access

Keys are scoped — your app only gets access to the data the user explicitly approved. Request the minimum scopes you need.

Rate Limiting

API calls are rate limited per key to ensure fair usage.

100 requests / minute
Auth Header Example
bash
# Every request needs this header
curl -H "Authorization: Bearer ic_YOUR_KEY" \
  https://us-central1-ironcore-f68c2.cloudfunctions.net/apiGetProfile
Python Example
python
import requests

BASE = "https://us-central1-ironcore-f68c2.cloudfunctions.net"
KEY  = "ic_YOUR_KEY"

headers = {"Authorization": f"Bearer {KEY}"}

profile = requests.get(
    f"{BASE}/apiGetProfile",
    headers=headers
).json()

print(profile["username"], "—", profile["league"])
